# Удаление информации о количестве документов из аккордеона

## Изменение
Убрана информация о количестве документов и количестве согласованных документов из заголовка аккордеона "Согласование документов".

## Что было изменено
В файле `src/components/ContractNegotiation.tsx`:

### Удалено:
1. **Свойства аккордеона:**
   - `badge={getDocumentApprovalStatus().status}`
   - `badgeColor={getDocumentApprovalStatus().color}`

2. **Функция `getDocumentApprovalStatus()`:**
   - Полностью удалена функция, которая формировала текст бейджа
   - Функция возвращала информацию вида "2/3 согласованы" или "Все согласованы"

### Результат:
Аккордеон "Согласование документов" теперь отображается с чистым заголовком без дополнительной информации о статусе документов.

## До изменения:
```
Согласование документов [2/3 согласованы] ▼
```

## После изменения:
```
Согласование документов ▼
```

## Статус
✅ **ВЫПОЛНЕНО** - Информация о количестве документов удалена из заголовка аккордеона

## Примечание
Функциональность автоматического сворачивания аккордеона после согласования всех документов сохранена, так как она использует функцию `isAllDocumentsApproved()`, которая не была изменена.
