# 📖 Руководство пользователя Microclimat Analyzer

## 🎯 Обзор системы

**Microclimat Analyzer** — это комплексная система для управления проектами квалификации климатических установок, анализа данных логгеров температуры и влажности, а также создания отчетов по результатам испытаний.

---

## 🚀 Основные модули системы

### 1. 📊 **Управление проектами**
**Доступ:** Все роли (кроме базового пользователя)

#### Функции:
- **Создание проектов квалификации**
- **Управление статусами проектов**
- **Назначение контрагентов и объектов квалификации**
- **Отслеживание прогресса выполнения**

#### Статусы проектов:
- `draft` - Черновик
- `contract_negotiation` - Согласование договора
- `testing_execution` - Проведение испытаний
- `creating_report` - Создание отчета
- `completed` - Завершен

---

### 2. 🏢 **Управление контрагентами**
**Доступ:** Все роли (кроме базового пользователя)

#### Функции:
- **Добавление новых контрагентов**
- **Редактирование информации о контрагентах**
- **Поиск и фильтрация контрагентов**
- **Управление контактной информацией**

---

### 3. ⚙️ **Управление оборудованием**
**Доступ:** Все роли (кроме базового пользователя)

#### Функции:
- **Каталог климатического оборудования**
- **Управление характеристиками оборудования**
- **Связывание оборудования с проектами**
- **Отслеживание использования оборудования**

---

### 4. 📈 **Анализатор временных рядов**
**Доступ:** Все роли (кроме базового пользователя)

#### Функции:
- **Загрузка файлов данных логгеров** (`.vi2`, `.xls`, `.xlsx`)
- **Автоматический парсинг данных** температуры и влажности
- **Визуализация данных** в виде графиков
- **Анализ соответствия критериям**
- **Экспорт результатов анализа**

#### Поддерживаемые форматы:
- **`.vi2`** - файлы логгеров Testo (174H, 174T)
- **`.xls/.xlsx`** - файлы Excel с данными измерений
- **`.csv`** - текстовые файлы с данными
- **`.pdf`** - документы

#### Плейсхолдеры в шаблоне отчета (DOCX):
В шаблоне отчета по испытаниям можно использовать подстановки: **`{Position}`** — должность пользователя из справочника пользователей; **`{Size}`** — только значение площади объекта (без типа помещения и зон хранения). Подробный список плейсхолдеров — в разделе **Справка** в системе.

### 6. 🔧 **Технические инструменты**
**Доступ:** Все роли (кроме базового пользователя)

## 📝 Пошаговые инструкции

### 🔐 **Вход в систему**

1. **Откройте браузер** и перейдите по адресу системы
2. **Введите email** и пароль
3. **Нажмите "Войти"**
4. **При первом входе** система может запросить смену пароля

#### Восстановление пароля:
1. **Нажмите "Забыли пароль?"** на странице входа
2. **Введите email** для восстановления
3. **Проверьте почту** и следуйте инструкциям
4. **Установите новый пароль**

---

### 📊 **Создание проекта квалификации**

1. **Перейдите в "Проекты"** в главном меню
2. **Нажмите "Создать проект"**
3. **Заполните обязательные поля:**
   - Название проекта
   - Контрагент
   - Описание
   - Дата начала
4. **Добавьте объекты квалификации**
5. **Нажмите "Сохранить"**

---

### 📁 **Загрузка и анализ данных логгеров**

#### Загрузка файлов:
1. **Откройте проект** в статусе "Проведение испытаний"
2. **Перейдите к объекту квалификации**
3. **В блоке "Снятие логгеров"** нажмите "Выбрать файл"
4. **Выберите файл** (`.vi2`, `.xls`, `.xlsx`)
5. **Дождитесь обработки** (прогресс-бар покажет статус)

#### Анализ данных:
1. **После загрузки** нажмите "Анализ данных"
2. **Просмотрите графики** температуры и влажности
3. **Настройте критерии** соответствия
4. **Экспортируйте результаты** при необходимости

---

### 📄 **Согласование документов**

1. **Перейдите в проект** в статусе "Согласование договора"
2. **Загрузите необходимые документы:**
   - Коммерческое предложение
   - Договор
   - Протоколы квалификации
3. **Добавьте комментарии** при необходимости
4. **Нажмите "Согласовано"** или "Отклонить"**
5. **Система автоматически** обновит статус проекта

---

### 👥 **Управление пользователями** (только для администраторов)

1. **Перейдите в "Пользователи"** в главном меню
2. **Нажмите "Добавить пользователя"**
3. **Заполните данные:**
   - ФИО
   - Email
   - Пароль
   - Роль
4. **Нажмите "Сохранить"**

#### Изменение роли пользователя:
1. **Найдите пользователя** в списке
2. **Нажмите "Редактировать"**
3. **Измените роль** в выпадающем списке
4. **Сохраните изменения**

## 🔍 **Диагностика и решение проблем**

### ❌ **"Доступ запрещен"**
**Причина:** Недостаточные права доступа
**Решение:**
1. Обратитесь к администратору для изменения роли
2. Выйдите и войдите в систему заново
3. Очистите кэш браузера

### 📁 **Файлы не загружаются**
**Причина:** Неподдерживаемый формат или поврежденный файл
**Решение:**
1. Проверьте формат файла (`.vi2`, `.xls`, `.xlsx`, `.csv`, `.pdf`)
2. Убедитесь, что размер файла не превышает 10MB
3. Попробуйте загрузить файл заново

### 🔄 **Данные не отображаются**
**Причина:** Проблемы с подключением к базе данных
**Решение:**
1. Проверьте интернет-соединение
2. Обновите страницу
3. Обратитесь к техническому специалисту

### 📊 **Ошибки парсинга данных**
**Причина:** Неправильный формат данных в файле
**Решение:**
1. Проверьте структуру файла
2. Убедитесь, что есть колонки с датой и температурой
3. Проверьте формат даты (DD.MM.YYYY HH:MM:SS)

## 🔒 **Безопасность и конфиденциальность**

### 🛡️ **Рекомендации по безопасности**
- **Используйте сложные пароли** (минимум 8 символов, заглавные и строчные буквы, цифры)

- **Регулярно обновляйте пароли**

### 🔐 **Политика конфиденциальности**
- Все данные хранятся в зашифрованном виде
- Доступ к данным имеют только авторизованные пользователи
- Все действия пользователей логируются
- Данные не передаются третьим лицам без согласия
